Backend

Worked as part of a team to develop a module in Java for a music playing application. Also used the Python based Django framework to build a news web app and the REST framework to support a JSON based web API. Developed a client for retrieving Hacker News headlines using Python and deployed release to PyPi.

Front End

On the frontend, I used jQuery with AJAX. Used Angular to build this web site and a nearby restaurant browsing app which integrates with Foursquare API.

Big Data

Hadoop and Spark to perform sentiment analysis on Twitter data, aggregations on financial data and clustering of Stack Overflow data. I have also performed network analysis on large graphs using iGraph on R; using measures such as centrality and modularity for graph comparison and Gephi for graph visualisation.

Database

Leveraged JDBC for persistence in Java application. Used MongoDB and Firebase Realtime Database. Knowledge of SQL, including experience working with Oracle, Postgres and SQLite.

Robotics

Worked with robotics for my undergraduate research project, using Gazebo for simulation and Robot Operating System (ROS) for actuator control. Unity3D and the Oculus Rift head-mounted display was then used to remotely control the robot.
